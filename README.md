- Node.js (v18 or later)
- npm, yarn, or pnpm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```

Now, open [http://localhost:9002](http://localhost:9002) in your browser to see the application. The main recipe generator can be found at `/generator`.
