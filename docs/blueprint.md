# **App Name**: Green Grocer Recipes

## Core Features:

- Ingredient Input: Allow users to input the ingredients they currently have available.
- Recipe Suggestion: Suggest recipes based on the user's ingredients, powered by AI. The AI tool will be responsible for suggesting new options.
- Recipe Display: Display the full recipe instructions in a clear, step-by-step format.
- Favorite Recipes: Allow users to save their favorite recipes for later viewing.

## Style Guidelines:

- Primary color: Fresh green (#90EE90) to evoke natural and healthy vibes.
- Background color: Soft white (#F0F8FF) for a clean and user-friendly design.
- Accent color: Light yellow-green (#ACF39D) for highlights and calls to action.
- Body and headline font: 'PT Sans' (sans-serif) for a modern look, suitable for both headlines and body text.
- Use clear, minimalist icons to represent different ingredients and recipe categories.
- Simple, clean layout with a focus on readability and ease of navigation.
- Subtle animations to provide feedback and enhance user interaction, like fading in recipe steps.
